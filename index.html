HTML TAGS and DESCRIPTION
SI. Tag Name Description Example
01 <!DOCTYPE> Defines the document type
02 <html> Defines an HTML document
03 <head> Contains metadata/information for the document
04 <title> Defines a title for the document
05 <body> Defines the document's body
06 <h1>to<h6> Defines HTML headings
07 <p> Defines a paragraph
08 <br> Inserts a single line break
09 <hr> Defines a thematic change in the content
10 <!--...--> Defines a comment
